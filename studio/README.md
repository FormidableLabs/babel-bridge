## Getting started

### Prerequisites

- [Node.js](https://nodejs.org/en/download) (`v18.0.0`+)
- [pnpm](https://pnpm.io/installation) (`v8.7.6`+)

### Installation

1. Install dependencies (`pnpm install`)
1. Run the studio (`pnpm dev`): http://localhost:3333
1. Have fun!
